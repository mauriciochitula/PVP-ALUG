Política de Privacidade

Este site é um projeto experimental da PVP e não representa o portal oficial da empresa. As informações aqui apresentadas têm caráter demonstrativo e podem incluir formulários ou simulações que não correspondem a serviços reais.

Coleta de Dados

Coletamos informações como nome, email, telefone e dados de navegação. Estes dados são usados para testes da plataforma e para garantir a segurança dos utilizadores.

Uso das Informações

As informações podem ser usadas para simular o funcionamento da plataforma e, em casos específicos, para localizar transgressores que tenham atitudes negativas no site.

Compartilhamento de Dados

Os dados não serão comercializados. No entanto, poderão ser compartilhados com autoridades policiais caso seja necessário para investigação de comportamentos ilegais.

Cookies e Tecnologias

Este projeto utiliza cookies para melhorar a navegação e analisar o uso do site. O utilizador pode configurar o navegador para recusar cookies, mas isso pode limitar funcionalidades.
